## Overview
This is a real-time chat application using Socket.io, React (Vite) frontend and an Express + Socket.io backend.

Features:
- Real-time messaging
- Multiple chat rooms
- Private messaging (direct messages)
- Typing indicators
- Read receipts (basic)
- Presence: online users list
